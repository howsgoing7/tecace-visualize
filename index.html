01 — Test Environment

Device & Model Setup

This benchmark places two Samsung flagship devices side-by-side with identical model weights, identical prompts, and identical conditions — the only variable is the hardware generation. The goal is to isolate the contribution of the SoC revision to on-device AI inference performance.

Device Specifications
Both devices run Android 15 with the same 8-core CPU configuration. The key difference lies in the SoC revision.

04 — Token Processing Speed

Decode TPS vs Prefill TPS

Token processing speed breaks down into two distinct phases: Prefill (processing the input prompt) and Decode (generating the output tokens). These are governed by different compute patterns — prefill is parallelizable and benefits from memory bandwidth; decode is sequential and more sensitive to per-core throughput.
